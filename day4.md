### Day-4: Script Exeucation Permission

```bash

chmod 755 /tmp/xfushion.sh
```

What does 755 mean?
Each digit refers to permissions for a category of users:
Digit
User Class
Permissions
7
Owner
Read (4) + Write (2) + Execute (1) = 7 (rwx)
5
Group
Read (4) + Execute (1) = 5 (r-x)
5
Others
Read (4) + Execute (1) = 5 (r-x)
